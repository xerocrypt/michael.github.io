Open Dashboard and Diagnostics Service

Context

This application was developed to display metrics for a clinical systems messaging service, to assist a database operations team in the detection and timely resolution of messaging errors. While ASP.NET MVC provides an excellent framework on which to build such a metrics dashboard, a considerably more lightweight alternative would be to develop it from JavaScript and Node.js.

Requirements

  • Node.js
Also assumes there are stored procedures for the data you wish to extract from the SQL server (MS SQL or MySQL). The code is to be customised for your installation.

Technical Overview

Is based on JavaScript and Node.js for the controllers and logic.
