SQLite 3 Commands

Create a new table: CREATE TABLE Introduction (Id INTEGER NOT NULL PRIMARY KEY AUTOINCREMENT, English TEXT, German TEXT, Note TEXT, Category TEXT);
Insert a record into a table: INSERT INTO Introduction VALUES (0001, 'Test', 'Test', '', 'Introduction');
Select all records from a table: SELECT * FROM Introduction
Display the schema of a table: .schema Introduction
Update an entry: UPDATE Introduction SET English = 'How are you?' WHERE Id = 2;
