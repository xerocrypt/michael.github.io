Raspberry Pi

These were really popular among the lecturers and high-grade students at university, probably because they were the cheapest and most convenient way of getting an actual physical Linux box. I never got my hands on one until now.



What I have here is a Raspberry Pi 3 (model B), which has a Broadcom quad-core processor, 1GB memory, four USB ports and HDMI.

As you can see, the Pi has a set of GPIO ports along the side for hardware and I/O programming. If you're into that sort of thing, you'd get a stronger foundation in computer science learning with a simple development board, such as the Arduino Uno. But I think it would primarily it would make a decent base system for a desktop.

The PiBow Tangerine case I bought from ModMyPi for Â£12, and it was nicely packaged. The construction is pretty solid and easy to assemble.

Okay, the Pi needs a few other things:
  • Power supply
  • Case: Orange case, available from ModMyPi.
  • MicroSD card with operating system
  • Keyboard
  • Mouse
  • Monitor
It looks as if the device will require the official power supply, as it's important to get one that's capable of supplying 2.5A.

I wouldn't recommend attaching anything fancy or running devices through multiple USB hubs, as that could draw enough current to fry part of the board. I've seen that happen with laptops a couple of times.

Operating System

One of the nice things about the Pi is it'll run whatever operating system happens to be on the inserted MicroSD card, so it's easy to switch between them. Alternatives to the recommended Raspbian OS include Debian, PwnPi and several media player setups. A colleague also showed me a pen testing installation called 'PwnPi'. An operating system can also be loaded from an attached USB drive or portable HDD.

While the official Ubuntu MATE site states that it's possible to get a usable installation on an 8GB card, but I strongly recommend using at least 16GB since developer toolchains, programming libraries, customisations and other applications will require several more GBs.
Another thing to remember, especially when running the OS from the microSD is the data transfer rate might be a bottleneck, and therefore a higher-performance microSDHC card is recommended.

It's not so much the Pi hardware itself that makes is an educational resource, but rather what you're doing with the operating system.
Now, the next thing you'll need is the Ubuntu MATE download specifically for the Pi. The download is a .xz archive - the .img file needs to be extracted from this, using something like 7-Zip. The image file must then be 'burned' to the SD card, so you'll need a program for that also. I'm using Etcher, which is recommended in the official Pi documentation.

I'll be installing Ubuntu MATE. This should run fine on 1GB memory, and it's quite possible to scrap the desktop manager and interface if I need to, and simply use the command line.
I'm giving this the hostname 'ORAC', after that annoying acrylic box of components off Blake's 7.
