What Developers Need to Know

Back in 2003/2004, during my college days, we had a module called Internet Technologies, taught by an accomplished Web developer who made a fair amount of money. All he needed to know, and all we needed to learn, was HTML, CSS and JavaScript.

Ten years later, in my first development job, I had to know a lot more. These days, it looks as if developers need to learn a lot more technologies than developers in the past. This was an awakening, from my undergraduate days during which I did functional programming. In fact, there are so many other considerations that a new specialisation, DevOps, became all the rage.
